A game using the This Could Be Better Game Framework is attempting to start.

If this message remains visible after a few seconds, it is likely that startup has failed. Usually this means that either JavaScript is disabled or blocked, or the web browser is not running with the settings and environment necessary to be able to read text files. Try the following:

  • In the web browser's settings, enable JavaScript.
  • Locate any script blockers running as web browser plugins and turn them off for this site, for this tab, or for all sites.
  • If hosting the game from your own computer, close all browser windows and restart using more relaxed file-security settings, for example, the --allow-file-access-from-files for the Google Chrome browser. A script to do this is provided in the _Scripts directory. (Note that this could, in theory, expose your browser to attack from malicious sites visited while the setting is enabled.)
